# HEX to RGB Converter

Конвертер цветов из HEX в RGB, сделанный на React.  
Позволяет вводить цвет в формате HEX (`#RRGGBB`), видеть RGB-значение и менять фон на выбранный цвет.  
При некорректном HEX выводится ошибка.
